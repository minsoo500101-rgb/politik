개인정보 처리방침

대한민국 패치노트(patchkr.com, 이하 "사이트")는 사용자의 개인정보를 중요하게 생각하며 다음 원칙에 따라 처리합니다.

최종 업데이트: 2026년 6월 24일 · 문의 minsoo500101@gmail.com

1. 수집하는 정보

사이트는 회원가입·로그인 기능이 없으며, 개인을 식별할 수 있는 정보를 직접 수집하지 않습니다.

  • 자동 수집 정보(분석용) — 방문 페이지·접속 시간·브라우저 종류·OS·기기 종류 등 익명 통계.
  • localStorage 캐시 — 사이트 데이터의 빠른 재로드를 위해 브라우저에 저장하며, 서버로 전송되지 않습니다.

2. 광고 (Google AdSense)

사이트는 Google AdSense를 통해 광고를 게재할 수 있으며, 이 과정에서 Google은 다음 정보를 처리할 수 있습니다.

  • 제3자 공급업체(Google 포함)는 쿠키를 사용해 사용자의 이전 방문 정보를 기반으로 광고를 게재합니다.
  • Google은 광고 쿠키(DoubleClick DART 쿠키 등)를 사용해 사용자가 이 사이트 및 다른 사이트를 방문한 기록을 기반으로 광고를 게재합니다.
  • 사용자는 Google 광고 설정에서 개인 맞춤 광고를 비활성화할 수 있습니다.
  • 사용자는 aboutads.info에서 제3자 광고 공급업체의 쿠키를 비활성화할 수 있습니다.
  • 사이트는 선거기간의 정치 광고 및 종교 광고를 차단합니다.

3. 제3자 서비스

사이트는 다음 외부 서비스의 API를 호출하여 데이터를 표시합니다. 이 과정에서 사용자의 IP 주소가 해당 서비스에 노출될 수 있습니다.

  • 대한민국 국회 OPEN API (open.assembly.go.kr)
  • 중앙선거관리위원회 (data.go.kr) — Vercel Serverless 프록시 경유 (사용자 IP 미노출)
  • 국가법령정보센터 (open.law.go.kr) — Vercel Serverless 프록시 경유
  • 한국어 위키백과 (ko.wikipedia.org) — 브라우저 직접 요청
  • Wikimedia 페이지뷰 API (wikimedia.org) — 브라우저 직접 요청
  • 네이버 검색 API (openapi.naver.com) — Vercel Serverless 프록시 경유

4. 정보 보유 기간

  • 사이트 서버는 사용자 식별 정보를 저장하지 않습니다.
  • 브라우저 localStorage 캐시는 일정 시간 후 자동 갱신됩니다.
  • Google AdSense 쿠키는 Google 정책에 따릅니다(통상 13개월).

5. 사용자 권리

  • 브라우저 설정에서 쿠키를 차단할 수 있습니다(광고 일부 기능이 제한될 수 있음).
  • 브라우저 설정에서 localStorage를 삭제할 수 있습니다.
  • Google 광고 설정에서 맞춤 광고를 비활성화할 수 있습니다.

6. 변경 사항

본 방침은 법령·서비스 변경에 따라 수정될 수 있으며, 변경 시 본 페이지에 공지합니다.
